## Moom 4 主要功能

**浮窗调整**：鼠标悬在窗口的控制按钮上，就会弹出一个可完全自定义的浮窗 Palette，里面放你常用的布局方案，悬停一下就能切换，不用拖不用点。这个 Palette 最多可以放 61 个方案，还支持文件夹嵌套，实际上没有数量限制。

**鼠标快捷缩放（Hover 功能）**：按住修饰键，把鼠标移到任何一个窗口上——就算那个窗口在后台——就可以直接拖动它的边缘来调整大小，不需要先点击激活它。听起来很小的功能，用上了真的很顺手。

**边缘拖拽分屏**：把窗口往屏幕边缘一拖，自动出现布局选项，全屏 / 半屏 / 四分之一屏任选。而且你还可以把自定义的布局方案绑到拖拽区域上，一个放下去就能触发整套窗口排列。

**键盘控制**：不爱用鼠标的人有福了。设好快捷键之后，呼出控制面板，用方向键就能移动和调整窗口大小，完全不用碰鼠标。

**保存与恢复布局**：把你惯用的窗口排列保存成方案，下次一键调出来，乱掉的桌面秒变整洁。对于要在外接显示器和笔记本之间切换的用户来说，这个功能非常省事。

**动作链（Chains）**：一个操作里串联多个动作——比如「把这个窗口移到另一个屏幕，然后调成指定大小」，一步完成。
